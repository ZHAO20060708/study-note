在二阶常系数齐次线性微分方程的求解中，当特征方程出现两个相等的实根（即重根）时，通解中第二项需要乘以 $x$，这并非人为规定的经验法则，而是为了**寻找第二个与原解线性无关的特解**所推导出的必然数学结果。

从严谨的数学角度，主要有两种经典的方法可以证明并解释这一现象：**降阶法（常数变易法）**与**算子求导法（极限法）**。

### 一、 问题背景定义

考虑标准的二阶常系数齐次线性微分方程：

$$a y'' + b y' + c y = 0 \quad (a \neq 0)$$

其对应的特征方程为：

$$a r^2 + b r + c = 0$$

当判别式 $\Delta = b^2 - 4ac = 0$ 时，特征方程有两个相等的实根 $r_1 = r_2 = r = -\frac{b}{2a}$。

此时，我们立刻可以得到一个特解：

$$y_1 = e^{rx}$$

由于是二阶微分方程，其通解必须由两个**线性无关**的特解 $y_1$ 和 $y_2$ 线性组合而成（即比值 $\frac{y_2}{y_1}$ 不能为常数）。由于只有一个根 $r$，单纯的 $e^{rx}$ 无法提供两个线性无关的解，因此我们需要寻找第二个解 $y_2$。

---

### 二、 证明方法 1：降阶法（达朗贝尔方法）

降阶法（Reduction of order）是构造第二个线性无关解的最通用、最严谨的方法。我们假设第二个解是第一个解的某个非常数函数倍数，即令：

$$y_2 = u(x) y_1 = u(x) e^{rx}$$

我们需要求出未知函数 $u(x)$。对 $y_2$ 求一阶和二阶导数：

$$y_2' = u' e^{rx} + u r e^{rx}$$

$$y_2'' = u'' e^{rx} + 2u' r e^{rx} + u r^2 e^{rx}$$

将 $y_2, y_2', y_2''$ 代入原微分方程 $a y'' + b y' + c y = 0$ 中：

$$a(u'' e^{rx} + 2u' r e^{rx} + u r^2 e^{rx}) + b(u' e^{rx} + u r e^{rx}) + c(u e^{rx}) = 0$$

由于 $e^{rx} \neq 0$，我们可以将其从方程中约去，并按 $u$ 的导数阶数重新合并同类项：

$$a u'' + (2ar + b) u' + (ar^2 + br + c) u = 0$$

根据已知条件分析上述方程的系数：

1. $r$ 是特征方程的根，因此 $ar^2 + br + c = 0$。
    
2. $r$ 是重根，其值为 $r = -\frac{b}{2a}$，因此推导出 $2ar + b = 0$。
    

代入这两个条件，原本复杂的微分方程极大地坍缩为：

$$a u'' = 0$$

因为 $a \neq 0$，所以：

$$u'' = 0$$

对 $u'' = 0$ 连续积分两次，得到：

$$u(x) = C_1 x + C_2$$

为了寻找一个最简单的、且与 $y_1$ 线性无关的特解，我们可以取 $C_1 = 1, C_2 = 0$（若 $C_1=0$，则解与 $y_1$ 线性相关），此时得到：

$$u(x) = x$$

因此，第二个线性无关的特解正是：

$$y_2 = x e^{rx}$$

这就是为什么在通解 $y = C_1 y_1 + C_2 y_2$ 中，必须多乘一个 $x$ 的根本原因。

---

### 三、 证明方法 2：微分算子求导法

另一种更为优雅的代数视角是引入微分算子 $L$：

$$L[y] = a y'' + b y' + c y$$

将试探解 $y = e^{rx}$ 作用于该算子，得到：

$$L[e^{rx}] = (ar^2 + br + c) e^{rx} = P(r) e^{rx}$$

其中 $P(r) = ar^2 + br + c$ 为特征多项式。

如果 $r_1$ 是重根，这意味着特征多项式可以写成 $P(r) = a(r - r_1)^2$。

对于重根，多项式及其一阶导数在 $r_1$ 处均等于零，即：

$$P(r_1) = 0, \quad P'(r_1) = 0$$

现在，我们对等式 $L[e^{rx}] = P(r) e^{rx}$ 两边关于参数 $r$ 求偏导数。因为求导关于 $x$ 和关于 $r$ 的操作是可交换的：

$$\frac{\partial}{\partial r} L[e^{rx}] = L\left[ \frac{\partial}{\partial r} e^{rx} \right] = L[x e^{rx}]$$

同时，对右侧应用乘积法则：

$$\frac{\partial}{\partial r} (P(r) e^{rx}) = P'(r) e^{rx} + P(r) x e^{rx}$$

将上述两者结合，得到一个新的算子恒等式：

$$L[x e^{rx}] = P'(r) e^{rx} + P(r) x e^{rx}$$

当我们把重根 $r = r_1$ 代入上述等式时，由于 $P(r_1) = 0$ 且 $P'(r_1) = 0$，等式右侧完全变为零：

$$L[x e^{r_1 x}] = 0 \cdot e^{r_1 x} + 0 \cdot x e^{r_1 x} = 0$$

这证明了 $y = x e^{r_1 x}$ 同样是使得方程 $L[y] = 0$ 成立的解。

---

### 总结

乘上 $x$ 并非定义出来的形式，而是由微积分基础定理严密推导得出的。当特征多项式出现多重根时，系统的自由度（方程阶数）要求有足够数量的独立解，而乘以 $x$（或者更高阶重根时乘以 $x^2, x^3$ 等）恰好弥补了根的简并带来的解空间的维度缺失。
