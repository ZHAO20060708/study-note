## 1. 阶 (Order) vs 次 (Degree)

在常微分方程（ODE）中，这两个词描述的是完全不同的维度：

### **阶 (Order)**

**定义：** 指方程中出现的**最高阶导数**的阶数。

- 如果方程里最高出现了 $y''$（二阶导），那它就是二阶微分方程。
    
- 通式：$F(x, y, y', \dots, y^{(n)}) = 0$ 是一个 **$n$ 阶**微分方程。
    

### **次 (Degree)**

**定义：** 当微分方程可以写成关于 $y$ 及其各阶导数的**多项式**形式时，**最高阶导数所在的幂次**称为该方程的“次”。

- 注意：如果方程里包含 $\sin(y')$、$e^{y''}$ 这种超越函数形式，通常不讨论“次”的概念，因为它们无法表示为多项式。
    

### **对比示例**

|**方程**|**最高阶导数**|**阶 (Order)**|**次 (Degree)**|
|---|---|---|---|
|$(y'')^3 + y' + y = 0$|$y''$|2|3|
|$y' + x^5 y = \sin(x)$|$y'$|1|1|
|$\sqrt{y''} + y = 0 \Rightarrow y'' = (-y)^2$|$y''$|2|1 (化简后)|

## 2. 为什么 $y'' + \sin(y) = 0$ 不是线性的？

判定一个微分方程是否为**线性 (Linear)**，标准非常严苛。一个 $n$ 阶线性微分方程必须能写成如下标准形式：

$$a_n(x) \frac{d^n y}{dx^n} + a_{n-1}(x) \frac{d^{n-1} y}{dx^{n-1}} + \dots + a_1(x) \frac{dy}{dx} + a_0(x)y = f(x)$$

### **线性判定的两个死理：**

1. **因变量及其导数必须是一次的：** $y, y', y'', \dots$ 每一个的幂次都必须是 $1$。
    
2. **系数只与自变量 $x$ 有关：** $y$ 及其导数的系数 $a_i(x)$ 不能包含 $y$ 或 $y$ 的任何导数。
    

### **对 $y'' + \sin(y) = 0$ 的处决：**

在这个方程中，项 $\sin(y)$ 是罪魁祸首。

- 从算子角度看：令 $L[y] = y'' + \sin(y)$。线性要求满足叠加原理，即 $L[cy] = cL[y]$。
    
- 显然，$\sin(cy) \neq c \sin(y)$（除非 $c=1$ 或 $0$）。
    
- 如果你把 $\sin(y)$ 展开成泰勒级数：$\sin(y) = y - \frac{y^3}{3!} + \frac{y^5}{5!} - \dots$，你会发现它包含了 $y$ 的三、五、七等无穷多高次幂。
    

**结论：** 只要因变量 $y$ 被包裹在非线性函数（如 $\sin, \cos, \ln, e^y$ 或直接是 $y^2$）里，这个方程就彻底告别“线性”了。这个具体的方程其实是**单摆方程**（无阻尼、大角度情况），是典型的非线性力学问题。
