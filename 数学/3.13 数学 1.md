## Key Concepts

### 1. 阶数 (Order) 与次数 (Degree)

- **阶数**由方程中出现的 **最高阶导数** 决定。
- **次数**是将最高阶导数项化为 **有理整式** 后，该最高阶导数的幂次。
- **示例：** $y''' + (y'')^5 = 0$
  - 最高阶导数为 $y'''$（三阶），其幂次为 $1$。
  - 故该方程为 **三阶一次** 方程。

### 2. 线性性判断

方程关于 $y, y', y'', \dots$ 必须同时满足以下条件才是线性的：

- 各导数项与 $y$ 本身的幂次均为 **一次**；
- 不同阶的导数项与 $y$ 之间 **不能交叉相乘**（如 $yy'$ 即为非线性项）；
- $y$ 及其各阶导数 **不能嵌套在非线性函数中**（如 $\sin y$、$e^y$ 均违规）。

### 3. 积分因子法

针对一阶线性方程的标准形式：

$$y' + P(x)\,y = Q(x)$$

- **积分因子：** $\mu(x) = e^{\int P(x)\,dx}$
- 两端同乘 $\mu(x)$ 后，左端恰化为 $\bigl(y \cdot \mu(x)\bigr)'$，从而可直接积分求通解。

### 4. 特征方程法

用于求解二阶常系数齐次线性方程：

$$ay'' + by' + cy = 0$$

写出特征方程 $ar^2 + br + c = 0$，根据判别式 $\Delta = b^2 - 4ac$ 分三种情况：

| 判别式 | 根的类型 | 通解形式 |
| :--- | :--- | :--- |
| $\Delta > 0$ | 两个不等实根 $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| $\Delta = 0$ | 重根 $r_1 = r_2 = r$ | $y = (C_1 + C_2 x)\,e^{rx}$ |
| $\Delta < 0$ | 共轭复根 $\alpha \pm \beta i$ | $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

### 5. 齐次代换

当方程中反复出现 $y/x$ 的组合时，令 $u = y/x$（即 $y = ux$），则：

$$y' = u + xu'$$

代入后原方程必然化为 **可分离变量** 的形式，再按标准流程求解即可。

## Vocabulary List

| 术语 | 定义 |
| :--- | :--- |
| **通解 (General Solution)** | 含有与方程阶数相同个数的、相互独立的任意常数 $C$ 的解。 |
| **特解 (Particular Solution)** | 由特定初始条件确定了所有常数值的解，或本身不含任意常数的解。 |
| **恰当方程 (Exact Equation)** | 形如 $M(x,y)\,dx + N(x,y)\,dy = 0$ 且满足 $\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}$ 的方程（左端恰为某函数的全微分）。 |
| **特征根 (Characteristic Root)** | 特征方程 $ar^2 + br + c = 0$ 的代数解，直接决定二阶线性方程通解的基底函数形式。 |
| **伯努利方程 (Bernoulli Equation)** | 形如 $y' + P(x)\,y = Q(x)\,y^n$（$n \neq 0,\,1$）的非线性方程；通过换元 $z = y^{1-n}$ 可化为一阶线性方程求解。 |
