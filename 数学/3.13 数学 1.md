**Title:** 微分方程特训通关手册

**Subject:** 高等数学 - 常微分方程 (ODE)

**Topics:**

- 微分方程基础概念（阶、次数、线性性）
- 一阶方程核心解法（变量分离、一阶线性、齐次方程）
- 二阶常系数线性方程（特征方程法）
- 解的结构与唯一性

**Summary:** 本指南旨在帮助学习者梳理常微分方程的核心知识体系。内容涵盖了从最基础的方程定义辨析，到一阶与二阶方程的标准求解流程。特别针对学习者在“线性判断”、“特征根选取”以及“初值问题处理”上的常见易错点进行了重点强化。

**Key Concepts:**

- **阶数 (Order) 与 次数 (Degree):** 阶数由最高阶导数决定。次数是最高阶导数项（化为有理整式后）的幂次。例如 $y''' + (y'')^5 = 0$ 中，最高阶导数是三阶 $y'''$，其幂次为 1，故而是三阶一次方程。
- **线性性判断:** 方程对 $y, y', y'', \dots$ 必须是一次的，且导数项与 $y$ 项不能交叉相乘（如 $yy'$ 不行），也不能嵌套在非线性函数（如 $\sin y, e^y$）中。
- **积分因子法:** 针对一阶线性方程 $y' + P(x)y = Q(x)$，积分因子为 $\mu(x) = e^{\int P(x)dx}$。左右同乘积分因子后，左边可化为 $(y \cdot \mu(x))'$，这是求通解的杀手锏。
- **特征方程法:** 解决二阶常系数齐次线性方程 $ay'' + by' + cy = 0$。写出特征方程 $ar^2 + br + c = 0$，根据判别式 $\Delta = b^2 - 4ac$ 的结果，通解对应三种情况：两个不相等实根、两个相等实根（重根）、一对共轭复根。
- **齐次代换:** 看到 $y/x$ 这种组合，果断令 $u = y/x$（即 $y = ux$）。转换后原方程必然变为可分离变量方程，这是降维打击的常用手段。

**Vocabulary List:**

- **通解 (General Solution):** 含有与方程阶数相同个数的、且相互独立的任意常数 $C$ 的解。
- **特解 (Particular Solution):** 满足特定初始条件（Initial Condition）从而确定了常数，或本身不含任意常数的解。
- **恰当方程 (Exact Equation):** 形式为 $M(x,y)dx + N(x,y)dy = 0$，且满足偏导数交叉相等条件 $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$ 的微分方程（左端恰为某函数的全微分）。
- **特征根 (Characteristic Root):** 特征方程 $ar^2 + br + c = 0$ 的代数解，直接决定了二阶线性方程解的基底函数长什么样。
- **伯努利方程 (Bernoulli Equation):** 形式为 $y' + P(x)y = Q(x)y^n$ ($n \neq 0, 1$) 的非线性方程，需通过换元操作 $z = y^{1-n}$ 强行线性化。
